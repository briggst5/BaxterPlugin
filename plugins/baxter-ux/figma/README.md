## Adding exports

1. Download or export from Figma (`.fig`, PDF, PNG snapshots, or Code Connect files).
2. Drop files into `figma/incoming/` (create the folder if needed).
3. Run `./scripts/sync-dls.sh` to promote `incoming/` contents into `figma/`.
4. Record the export date and Figma version below.
