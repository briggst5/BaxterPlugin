### Presets

These presets are designed for immediate use, highlighting the component's adaptability. Designers can easily detach the component to develop more complex variants if needed, allowing for greater customization and flexibility in design. At least two segments are needed. See Guidelines on Number of Steps.







### Label

Use the Label boolean property to add a label to indicate the current step.
Even without a label, this component can serve as a valuable visual cue, helping users in understanding the state of completion of the process that is underway.







### Current Step

Use the Current Step property to set this variant. This component gives a visual indication of the state of completion, informing users of their precise advancement in a process. The current step is highlighted using the component/stepper/fill-current color token, that becomes the accent blue when done.
