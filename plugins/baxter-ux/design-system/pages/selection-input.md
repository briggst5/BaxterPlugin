### Filled

The input field of the component can either be presented as empty or have a default value. When the user changes the selection, regardless the initial condition, the new value is highlighted by switching text color from secondary to primary.

### Menu Items List

A Selection Input enables users to enter a value by selecting an option from a dropdown list. The dropdown list is displayed when transitioning from the closed to the open state. Users can toggle between these two states by tapping or clicking the component's input field. The touch area encompasses the entire input field, not just the trailing chevron. In the open state, the trailing chevron points upward while, in the closed state, it points downward.
