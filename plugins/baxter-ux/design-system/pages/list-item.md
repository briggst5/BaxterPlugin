### Variants

List items can be categorized into interactive and static types. Interactive list items are fully interactive, meaning the entire list item acts as a target area. In contrast, static items are generally not interactive; their interactivity is limited to the trailing item, which, if it is an interactive component (e.g., a chip), retains the properties of that component.

#### Divider Boolean

A list item can be set to display a divider. The designer has the discretion to determine whether the list should include dividers. It is essential that all items within a list block consistently either have a divider or do not have one (with the exception of the last list item).
