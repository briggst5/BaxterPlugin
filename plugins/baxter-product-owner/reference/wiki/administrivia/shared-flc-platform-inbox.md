There is a shared inbox available for use when exchanging information that should be tied to an individual.  The account is: `Global_FLC_Platform@baxter.com`

## Instructions for 'Sending As' the Shared Account

The IT instructions (above) also include instructions for sending messages as the shared account, and again, newer versions of Outlook are slightly different.  The instructions from IT show the "Options" ribbon fully visible in the New Email window.  Instead, the default behavior is that the ribbon is collapsed, and you will need to click on the ellipses to expand it:

![ad.png](/.attachments/ad-b0e00ea1-8657-4994-af7a-750695fcccd2.png =200x)

The rest of the steps in the instructions are consistent with recent Outlook versions.


## Controlling Access to the Shared Inbox

IT allows users who are designated as the mailbox owners to control the access list of the shared inbox.  The provided link is: https://coreapps.global.baxter.com/SharedMailBox/

![ae.png](/.attachments/ae-7a03fae5-9926-4053-8072-2769f5403e9a.png =350x)

The user interface allows adding and removing users to the shared inbox, as well as updating users with manager role over the inbox.

Then, in Outlook, use these steps to change individuals' access:
1.Access the shared inbox "Data File Properties...":
![af.png](/.attachments/af-bc4d3ad5-075b-46a8-95dc-bd151059b1a4.png =250x)


1. Edit the Mailbox Permission:
![ag.png](/.attachments/ag-9d9211d5-f757-4791-8352-4313550414df.png =200x)

1. Use the dialog to add / remove individuals.
![ah.png](/.attachments/ah-6f3cab65-f9e7-4532-8349-d393721fc113.png =200x)

- Normal users: Grant "Author" access to allow reading, sending, and deleting email
- Admin users: Grant "Owner" access to allow all actions, including changing security settings.
