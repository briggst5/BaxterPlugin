# Polarion

DHF artifacts (such as plans, requirements, and test artifacts) for the FLC Software Platform are developed and maintained in [Polarion](https://polarion.hrc.corp/polarion/#/project/flc.platform.01/home) before they are exported to SAP as the official system of record for approval and storing.

Polarion is a a web-based Application Lifecycle Management (ALM) platform designed to manage the entire software development lifecycle, from requirements and coding to testing and release. It provides a unified environment for teams to collaborate, ensuring traceability, compliance, and quality assurance in complex software and systems engineering projects.

If you do not have access to Polarion or the FLC Software Project within Polarion, request access using the [IT Now Platform](https://baxter.service-now.com/sp).  

Search "Polarion" and submit a Polarion Access Request.
